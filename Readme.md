# 💸 Project: Spendly: Finance Management Dashboard

## 📝 Overview

A powerful, intuitive finance management dashboard designed to help users track and manage their income and expenses seamlessly. With real-time updates, data visualizations, and user-friendly features, this tool provides clear insights into financial habits and supports smarter decision-making.

## 🚀 Tech Stack

- **Frontend**: React.js, TypeScript, TailwindCSS, shadcn/ui
- **Backend**: Django, Python
- **State Management**: Redux
- **Authentication & Database**: Firebase
- **Charts & Visualization**: Custom charts using libraries for pie, bar, and combined views

## 🌟 Features

- ✅ **Suggestion Warnings**

  - Positive and cautionary suggestions based on user transactions

- 🧑‍💼 **Custom Profile Bar**

  - Personalized profile section for each user

- 📊 **Transaction Table**

  - Add, edit, and delete options for managing transactions

- 🧩 **Advanced Filters**

  - Filter transactions based on various parameters for better insights

- 🥧 **Charts for Insights**

  - Pie chart for income vs. expenses
  - Combined charts with area and bar views for comprehensive analytics

- 🔒 **Authentication**
  - Secure login feature powered by Firebase Auth
